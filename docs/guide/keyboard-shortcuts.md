# Keyboard Shortcuts

Nav0 ships with 21 keyboard shortcuts covering tabs, navigation, bookmarks, zoom, and developer tools. Every shortcut is fully customizable.

## Quick Reference

In the tables below, **Mod** refers to **Cmd** on macOS and **Ctrl** on Windows/Linux.

### Tabs

| Action            | Default Shortcut | Description                 |
| ----------------- | ---------------- | --------------------------- |
| New Tab           | Mod+T            | Open a new empty tab        |
| Close Tab         | Mod+W            | Close the current tab       |
| Reopen Closed Tab | Mod+Shift+T      | Restore the last closed tab |

### Windows

| Action         | Default Shortcut | Description                        |
| -------------- | ---------------- | ---------------------------------- |
| New Window     | Mod+N            | Open a new browser window          |
| Private Window | Mod+Shift+N      | Open a new private browsing window |

### Navigation

| Action        | Default Shortcut | Description                 |
| ------------- | ---------------- | --------------------------- |
| Focus URL Bar | Mod+L            | Jump to the address bar     |
| Reload Page   | Mod+R            | Reload the current page     |
| Hard Reload   | Mod+Shift+R      | Clear site cache and reload |
| Back          | Alt+Left         | Navigate back in history    |
| Forward       | Alt+Right        | Navigate forward in history |

### Utilities

| Action          | Default Shortcut | Description                                                       |
| --------------- | ---------------- | ----------------------------------------------------------------- |
| Find on Page    | Mod+F            | Open find bar with regex, case-sensitive, and whole word matching |
| Command Palette | Mod+K            | Search tabs, bookmarks, history, downloads, and the web           |
| Open Settings   | Mod+,            | Open browser settings                                             |
| Open Downloads  | Mod+Shift+D      | Open the downloads page                                           |
| Open History    | Mod+Shift+H      | Open the history page                                             |

### Bookmarks

| Action         | Default Shortcut | Description                             |
| -------------- | ---------------- | --------------------------------------- |
| Bookmark Page  | Mod+D            | Bookmark or unbookmark the current page |
| Open Bookmarks | Mod+Shift+B      | Open the bookmarks page                 |

### View

| Action     | Default Shortcut | Description        |
| ---------- | ---------------- | ------------------ |
| Zoom In    | Mod+Shift+=      | Increase page zoom |
| Zoom Out   | Mod+Shift+-      | Decrease page zoom |
| Reset Zoom | Mod+0            | Reset zoom to 100% |

### Developer

| Action          | Default Shortcut | Description                          |
| --------------- | ---------------- | ------------------------------------ |
| Toggle DevTools | Mod+Shift+I      | Open or close Chrome Developer Tools |

## Customizing Shortcuts

All shortcuts are remappable:

1. Open **Settings** (Mod+,)
2. Navigate to the **Keyboard Shortcuts** section
3. Click on any shortcut to edit it
4. Press your desired key combination
5. The new shortcut takes effect immediately

Your customizations are stored locally and persist across browser restarts. To reset a shortcut to its default, clear the custom binding.

### Shortcut Format

Shortcuts use a modifier-key format:

- **Mod** — Cmd on macOS, Ctrl on Windows/Linux (automatically mapped per platform)
- **Shift** — Shift key
- **Alt** — Alt/Option key
- **+** — Separator between keys

For example, `Mod+Shift+T` means Cmd+Shift+T on macOS or Ctrl+Shift+T on Windows/Linux.

## Frequently Asked Questions

### Can I use the same shortcut for multiple actions?

No. Each key combination can only be bound to one action. If you try to assign a shortcut that is already in use, you'll need to clear the existing binding first.

### Do custom shortcuts sync across devices?

No. Nav0 does not sync any data. Keyboard shortcut customizations are stored locally on each device.

### Can I add entirely new shortcuts?

Currently, you can only remap the 21 built-in shortcut actions. Custom actions and macros are not supported.

### What if I break something with custom shortcuts?

You can reset any individual shortcut to its default, or reset all shortcuts by clearing your keyboard shortcuts settings.
