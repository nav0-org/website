Build from source

For maximum trust, or if you'd like to hack on Nav0. Clone, install, run. All three platforms are supported.

Why build it yourself?

You don't have to take our word for it. Reproducing the binaries from source is the strongest privacy guarantee we can offer. The full build is hermetic and the output is byte-equivalent to the published installers.

  • node 22.0 or newer
  • npm 10+
  • ~3 GB free disk
  • Roughly 4 minutes on an M1
~/projects — bash
$ git clone https://github.com/nav0-org/nav0-browser.git
$ cd nav0-browser
$ npm install
$ npm run make
→ Built artifact at out/Nav0-darwin-arm64/Nav0.app

The built app lands in the out/ directory. See the README for cross-compilation flags and code-signing details.
