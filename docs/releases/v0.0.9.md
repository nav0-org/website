# v0.0.9

<div class="release-date-meta">March 19, 2026</div>

## SSL Certificate Indicator

- **SSL Certificate Info Overlay** — click the lock icon in the address bar to view certificate details in a clean overlay panel
- **SSL Icon State** — address bar icon reflects whether the connection is secure, insecure, or has no certificate info
- **Overlay UX** — overlay closes on click-outside or Escape, auto-fits height, and avoids blur/click race conditions

## Fullscreen Fixes

- **Fullscreen Exit Toggle** — fixed fullscreen exit not working by tracking desired state instead of a one-shot flag
- **OS Fullscreen Respect** — stopped fighting macOS fullscreen exit; always respects the `leave-full-screen` event
- **Resize & Bounds** — fixed CSS layout, resize handling, and window/tab bounds when exiting fullscreen

## Drag & Drop Tabs Between Windows

- **Command O Cross-Window Tabs** — drag and drop tabs between windows directly from the Cmd+O tab switcher overlay

## Installer Improvements

- **Graceful Quit Prompt** — installer now prompts the user to close Nav0 instead of force-quitting
- **Silent Failure Fix** — fixed install script silently failing when Nav0 is already running
