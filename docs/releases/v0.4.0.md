# v0.4.0

<span class="release-badge latest">Latest</span>

<div class="release-date-meta">June 30, 2026</div>

A release that refines the tab bar and toolbar and fixes camera and microphone capture in packaged macOS builds.

## Improvements

- **Content-sized tabs** — tabs now size to their content (min-width down to 72px) with tighter, symmetric favicon spacing, restoring the fill-the-bar model so the strip never looks sparse
- **Hover-only close button** — the tab close (×) button is now hover-only, keeping inactive tabs clean and uncluttered
- **Clearer scroll chevrons** — the tab-strip scroll chevrons are larger (18px) and more prominent, matching the new-tab "+" glyph
- **Brand-red icon hover** — nav-bar and address-bar icon buttons now tint Nav0 brand-red on hover; disabled back/forward and the SSL indicator are excluded so they keep their own state colours

## Bug Fixes

- **Camera & microphone in packaged macOS builds** — packaged macOS builds are now code-signed (Developer ID + notarization when CI secrets are present, otherwise a valid ad-hoc signature) with hardened-runtime camera/microphone entitlements, so macOS TCC can attribute capture requests to Nav0
- **Native macOS capture prompt** — the app now drives the native macOS camera/mic permission prompt itself for `getUserMedia` requests from a tab, so granting the in-app permission actually delivers frames and audio (no-op on Windows/Linux)
