# v0.1.2

<span class="release-badge latest">Latest</span>

<div class="release-date-meta">April 4, 2026</div>

## App Rename: nav0-browser → Nav0

- **Package Identity** — renamed the application from `nav0-browser` to `Nav0` across package.json and package-lock.json
- **Installer Naming** — updated Squirrel (Windows) installer configuration in forge.config.ts to use the `Nav0` name
- **Install Script** — updated the macOS/Linux curl install script to reference the new `Nav0` binary name
- **CI/CD Pipeline** — updated GitHub Actions build workflow release notes and Homebrew cask references to use `Nav0`
- **Download Page** — updated all download filenames on the website to reflect the `Nav0` name
- **Performance Tests** — updated Puppeteer test binary paths to point to the renamed `Nav0` application
- **Documentation** — updated all documentation references from `nav0-browser` to `Nav0`, including the getting started guide, FAQ, and blog posts

## Note

GitHub repository URLs (`nav0-org/nav0-browser`) remain unchanged. Only the application name, binary references, and user-facing text were updated.
