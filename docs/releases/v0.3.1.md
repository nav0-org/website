# v0.3.1

<div class="release-date-meta">May 27, 2026</div>

A maintenance release focused on tab handling, session restore, and ad-blocker accuracy.

## Tabs & Navigation

- **Sub-frame navigations** — the address bar no longer updates from in-page navigations inside sub-frames, so the URL stays accurate to the page you're actually on
- **Helper popups** — Google API (gapi) helper frames and other invisible auth/helper popups no longer spawn stray empty tabs
- **Popups calling `window.open`** — `window.open` calls made from inside a popup window are now handled correctly

## Session Restore

- **Tab order** — tab order is now preserved exactly when restoring a previous session
- **Pinned tabs** — pinned tabs stay pinned across restarts

## Ad Blocker

- **First-party exemption** — the first-party / navigation exemption is now applied across all blocklists, preventing over-blocking
- **Tracker-list domains** — navigating to a domain that merely appears on a tracker list is no longer incorrectly blocked, fixing pages that wouldn't load

## Settings

- **Ad-blocker toggles** — toggles are now right-aligned inline with their labels
- **Permissions & forms** — corrected permission favicon sizing and tidied the allow-list layout and custom search-engine form

## Bug Fixes

- **Duplicate downloads** — downloads with a duplicate filename are now suffixed (e.g. `file (1).pdf`) instead of overwriting the existing file

## Improvements

- **Tab-strip resizing** — smoother tab-strip resizing by deferring the scroll `ResizeObserver` to the next frame
