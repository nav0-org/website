# v0.1.0

<span class="release-badge latest">Latest</span>

<div class="release-date-meta">March 29, 2026</div>

## Renderer Process Consolidation

- **8 Overlay Views → 2** — consolidated overlay renderer processes from 8 separate views down to 2, reducing memory usage and improving performance

## Tab Hibernation

- **Hibernate Inactive Tabs** — tabs inactive for 72 hours are automatically hibernated to conserve system resources

## Find in Page Improvements

- **Per-Tab Find State** — find-in-page state is now preserved per tab and maintained across tab switches
- **Initial Focus Fix** — find bar now correctly receives focus when opened
- **Tab-Specific Strips** — find bar and permission strips are now scoped to individual tabs with matching heights

## Web Share API Support

- **navigator.share Polyfill** — added Web Share API polyfill so websites using `navigator.share()` work correctly in Nav0

## UI & Window Chrome

- **Merged Title Bar** — hidden window title bar with controls merged directly into the tab bar for a cleaner look
- **Favicon Sizing** — fixed favicon sizing inconsistencies across tabs
- **Dark Mode Sync** — improved dark mode synchronization across the browser UI
- **Menu Restructuring** — reorganized browser menus for better discoverability
- **Options Menu Shortcuts** — keycap-styled text labels for keyboard shortcuts with wider menu to prevent text wrapping

## Branding

- **Nav0 Capitalization** — consistent Nav0 branding across the entire application
- **App Naming** — corrected app name display throughout the browser

## Bug Fixes

- **Per-Tab Permission Prompts** — permission prompts now work concurrently across multiple tabs without conflicts
- **SSL Dismiss** — fixed SSL info overlay dismiss behavior
- **Search Icon Sizing** — fixed oversized search icons in Downloads, History, and Bookmarks pages
- **Windows Download URL** — fixed download URL for Windows replacing `%20` with dot in exe filename
- **Fullscreen Traffic Lights** — removed extra padding for traffic light controls in fullscreen mode
