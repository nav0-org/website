# v0.2.9

<span class="release-badge latest">Latest</span>

<div class="release-date-meta">May 17, 2026</div>

## Address Bar

- **Autocomplete dropdown** — typing in the URL bar now surfaces a unified suggestions panel
- **Bookmarks over history** — autocomplete prefers bookmark matches over history hits for more deliberate results
- **Flattened results** — suggestions are presented as a single ranked list rather than grouped sections
- **Unified overlay** — autocomplete is rendered through the same overlay pipeline as other panels, fixing visibility and bounds issues

## Shortcuts

- **Focus address bar** — `Cmd/Ctrl+L` now focuses the URL input
- **Close tab on Linux/Windows** — `Ctrl+W` now closes the active tab instead of the entire window

## Private Mode

- **In-memory database** — the private browsing SQLite database now runs entirely in memory, so nothing touches disk
- **Leftover data cleanup** — any leftover private-mode data from previous sessions is now cleared on startup

## Bug Fixes

- **Ad blocker false positives** — generic URL patterns are no longer applied to first-party requests, reducing broken pages on legitimate sites
- **About page** — the integrity checksum row is hidden when no value is available, instead of showing an empty field
