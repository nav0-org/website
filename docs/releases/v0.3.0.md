# v0.3.0

<span class="release-badge latest">Latest</span>

<div class="release-date-meta">May 24, 2026</div>

The 0.3.0 release lands the Nav0 editorial redesign across every built-in page and tightens up address-bar, permissions, and private-mode behaviour that landed late in the 0.2.x line.

## Editorial Redesign

- **Tab strip and nav bar** — rebuilt around warm chrome surfaces (`--chrome-1` / `--chrome-2`), paper-white content, and a 2-px brand-red anchor on the active row
- **New tab page** — leads with an inline Command-K search field, plus Bookmarks and Frequently-visited tile rows on a shared card elevation
- **Bookmarks, History, Downloads** — redesigned with aligned filter pills, consistent row badges, and a unified toolbar
- **Settings** — redesigned around the editorial dashboard skeleton, with the unwired rebind UI removed and shortcut filters restyled
- **About page** — applies the design handoff
- **Options menu, Command-K, URL autocomplete** — share the same row highlight (`--bg-3` + red anchor) and tighter chrome offsets
- **Error page and SSL warning** — adopt the editorial grammar used across the rest of the chrome
- **Internal-page mastheads** — picked up a brand-red divider tint and dropped the sidebar brand lockup and section subtitles for a leaner read
- **DESIGN.md** — now the authoritative reference for tokens, components, iconography, motion, and the dashboard skeleton; CLAUDE.md points at it for any change under `src/renderer/`

## Address Bar

- **Enter without a highlighted suggestion** — now navigates to the URL you typed instead of swallowing the input
- **Arrow-key cycle** — includes the unselected state, so Up/Down can return to the typed query without jumping past it

## Permissions

- **Concurrent getUserMedia** — coalesced so the Google Meet camera prompt appears reliably when both audio and video are requested in quick succession

## Private Mode

- **Streaming sub-resources** — no longer blocked by URL-pattern filtering, fixing video seeking on common streaming sites in private windows
- **Autocomplete dismissal** — tightened so URL autocomplete closes before any modal overlay panel opens

## Developer Experience

- **Seed scripts** — new scripts populate a dev database and a saved session, anchored to `new Date()` with randomization for repeatable testing
- **CLAUDE.md** — refreshed against the current codebase layout

## Upgrade Notes

No migrations or settings changes are required. Existing installs pick up the new chrome on first launch.
